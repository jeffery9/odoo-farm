核心功能 | Key Features

设备全生命周期管理 | Complete Device Lifecycle Management

从设备配置、部署、监控到维护和退役的全流程管理。实现设备从采购到退役的全程跟踪。
Complete lifecycle management from device configuration, deployment, monitoring, to maintenance and retirement. Achieve full tracking from procurement to retirement.

实时遥测数据处理 | Real-time Telemetry Processing

高效处理来自设备的遥测数据，支持JSONPath声明式规则映射到业务对象。实现设备数据的实时分析和处理。
Efficiently process telemetry data from devices using JSONPath declarative rules to map to business objects. Achieve real-time analysis and processing of device data.

远程控制与OTA更新 | Remote Control & OTA Updates

通过安全的命令模板远程控制设备，支持固件OTA更新。实现远程设备管理和固件升级。
Remotely control devices through secure command templates, supporting firmware OTA updates. Achieve remote device management and firmware upgrades.

高级特性 | Advanced Features

安全可靠的MQTT桥 | Secure MQTT Bridge

基于FastAPI的高性能MQTT网关，提供HTTP-to-MQTT桥接。确保设备与Odoo系统间的安全稳定通信。
High-performance MQTT gateway based on FastAPI providing HTTP-to-MQTT bridging. Ensures secure and stable communication between devices and Odoo system.

Jinja2模板化命令 | Jinja2 Templated Commands

使用Jinja2模板引擎构建灵活的命令消息格式。支持动态参数和条件逻辑，实现复杂的设备控制。
Builds flexible command message formats using Jinja2 template engine. Supports dynamic parameters and conditional logic for complex device control.

智能数据映射 | Intelligent Data Mapping

基于JSONPath表达式的智能数据映射，支持域过滤和自动更新业务对象。实现设备数据与业务系统的无缝集成。
Intelligent data mapping using JSONPath expressions supporting domain filtering and automatic updating of business objects. Achieves seamless integration between device data and business systems.
